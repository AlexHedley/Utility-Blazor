## Features

- Url Encode
- Url
- Guid
- SQL (IN Clause / LIKE)
- Luhn
- Hidden Char Finder ([Original](https://www.soscisurvey.de/tools/view-chars.php) [Source](https://github.com/BurninLeo/see-non-printable-characters/blob/main/view-chars.php))
- HTML
- Markdown
- Ascii (WIP)
- HEX to RGB
- Binary
- String Convert
- Epoch Converter
- Remove whitespace
- Monaco Editor
- kb - mb - gb converter
- time converter
- Base64 Encode/Decode
